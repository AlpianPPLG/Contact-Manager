# Contact Manager App

Ini adalah aplikasi manajemen kontak yang dibangun menggunakan React dan Tailwind CSS.

## Fitur

- Menambahkan kontak baru
- Menampilkan daftar kontak
- Setiap kontak berisi nama, email, dan nomor telepon

## Instalasi

1. Clone repositori ini:

        git clone https://github.com/username/my-contact-manager.git


        cd my-contact-manager



2. Install dependencies

        npm install

3. Jalankan aplikasi

        npm start
